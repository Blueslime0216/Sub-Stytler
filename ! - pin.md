 # [ 깃허브 브렌치 ]
- main : 기본적인 버전 하나 [ x.x.0 ]
- hotfix : 버그 수정 브랜치 [ 0.0.x ]
- feature : 어떤 기능이 다 구현된 브랜치. feature가 쌓이면 release에서 완성된 버전 하나를 배포함
- WIP : Work In Progress, 작업 중인 브랜치. 기능이 완성되면 Feature로 옮겨감
