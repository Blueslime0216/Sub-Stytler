## 소개
Project.Sub-Stytler(가칭)는 유튜브 자막(.ytt)를 쉽게 제작할 수 있는 프로그램입니다.  


## 기능
- [ ] **자막 제작** : .ytt 자막을 제작할 수 있는 기본적인 기능들
- [ ] **자막 스타일** : 글자 색, 글자 크기, 그림자 추가 및 색상 변경 등
- [ ] **위치 애니메이션** : 자막 위치를 자유롭게 정할 수 있는 기능
- [ ] **이징** : 애니메이션에 이징을 넣을 수 있는 기능
- [ ] **번역** : 생성한 자막을 자동으로 번역할 수 있는 기능
- [ ] **트래킹** : 자막이 영상의 특정 영역을 따라다니는 기능
- [ ] **효과** : 자막에 유연하게 적용 가능한 효과들
- [ ] **다국어 지원** : 한국어, 영어, 일본어, 중국어 등의 언어를 지원할 예정입니다.
- [ ] **조절 가능한 UI** : 다양한 기능을 담당하는 각 UI를 원하는 대로 조정할 수 있게 할 예정입니다.
- [ ] **PC/Web 버전** : 로컬 PC에 설치할 수도 있고, 웹에서 바로 사용할 수도 있게 제작할 예정입니다.
- [ ] **비디오로 렌더(미정)** : 자막이 적용된 영상을 바로 렌더링 할 수 있는 기능. 영상에 자막이 추가된 상태로 비디오로 추출.

## 단축키 규칙
- [ 작업 특화 / 직관적 ] 등의 단축키 프리셋을 만들어두자
- 자막 프로그램이니깐 일반적인 타자 상태에서 쉽게 누를 수 있게 단축키를 설계 하자
- 마우스를 안 사용해도 작업할 수 있게 하자
- **Ctrl** : 기본적으로 단축키 앞에 붙음 
- **Shift** : 기존 기능과 비슷한 기능일때. "[Ctrl + S] 저장"과 "[Ctrl + Shift + S] 다른 이름으로 저장"
- **Alt** : 기존 기능의 반전. "[Ctrl + Z] 실행 취소"와 "[Ctrl + Alt + Z] 재실행"
